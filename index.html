What this extension does

This tool helps you analyze FIFA resale seats that are loaded in your browser. It supports Bosnian by default, can switch between Bosnian and English, and includes section search, category filters, “Best 100s”, and a safer category / section loading workflow.

Current features

  • Find cheapest loaded seats and best deals faster
  • Category filters with simplified categories 1–4 + Other
  • Section / block search
  • Best 100s summary for blocks in the 100–199 range
  • Bosnian-first popup with English toggle
  • Section discovery and controlled loading tools
  • Export loaded seats to CSV

Important limitation

The extension is designed to work with what FIFA loads in the page and with controlled section loading. If every section is not loaded, the “lowest” value is the lowest loaded so far, not always a guaranteed full-inventory lowest.

How to import into Chrome

  1. Download the ZIP from this page.
  2. Extract the ZIP to a normal folder on your computer.
  3. Open Chrome and go to chrome://extensions.
  4. Turn on Developer mode in the top-right corner.
  5. Click Load unpacked.
  6. Select the extracted folder that contains manifest.json.
Important:

Do not select the ZIP directly. You must extract it first, then choose the folder that has manifest.json, popup.html, background.js, and the icons folder.

How to configure and use it

First use

  1. Open the FIFA resale seat map page in Chrome.
  2. Click around sections on the map so FIFA loads seat data.
  3. Open the extension popup.
  4. Use the language toggle to switch between Bosnian and English if needed.

Recommended workflow

  1. Open a match seat map.
  2. Click around some sections manually first.
  3. Open the extension and click Discover Sections.
  4. Use category filters to focus on the seat category you care about.
  5. Use Load this category or Load All Discovered as needed.
  6. Use section search to narrow the popup to a specific block or area.
  7. Export to CSV if you want to compare later.

What the key fields mean

  • Cheapest: lowest currently loaded price
  • Best 100s: cheapest loaded seat in section / block 100–199
  • Category coverage: how complete the currently selected category appears to be
  • Overall coverage: how many discovered sections have been loaded so far

Troubleshooting

No data appears

  • Make sure you are on the FIFA seat map page, not just the main tickets page
  • Click around the map first so FIFA loads data
  • Reload the page once after loading the extension

Category click acts strangely

  • Category tabs should only filter
  • Use Load this category only when you want more sections for that category

Coverage looks incomplete

  • This means not all discovered sections have returned seat data yet
  • Use Load All Discovered or keep browsing the seat map manually
